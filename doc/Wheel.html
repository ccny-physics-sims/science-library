Class: Wheel

Wheel

new Wheel(x, y, d)

Makes a wheel that can display various forces while rotating or rolling.
Parameters:
Name Type Description
x number The x coordinate of the wheel's center.
y number The y coordinate of the wheel's center.
d number The wheel's diameter.
Properties:
Name Type Description
arrowDecorations array Customize the Arrow type and locations. Example: [{type: 'velocity', location_radial: 1, rimPos: 0 }]
arrows array Array of Arrow objects to display on the wheel.
rimColor color Color of the wheel rim.
spokeColor color Color of the wheel spokes.
wheelColor color Color of the rest of the wheel.
rotate bool Allows the wheel to rotate. (default: false)
angle num Angle in radians.
ang_speed num Angular speed.
trans_speed num Translational speed.
Wheel.addDecorations(decorations) method Draw the Arrow vectors.
Wheel.draw() method Draw the Wheel object.
Wheel.update() method Update the Wheel object.
