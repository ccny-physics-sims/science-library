Class: Graph

Graph

new Graph(w, h, x_min, x_max, y_min, y_max, resolution)

Constructs a Graph. Plot objects consisting of Point objects can be added and displayed on the Graph.
Parameters:
Name Type Description
w number Width of the Graph.
h number Height of the Graph.
x_min number Smallest value of the x-axis.
x_max number Largest value of the x-axis.
y_min number Smallest value of the y-axis.
y_max number Largest value of the y-axis.
resolution number The resolution of the Graph. Determines tick intervals.
Properties:
Name Type Description
title string Title of the graph.
xlabel string Caption for the x-axis.
ylabel string Caption for the y-axis.
showLabels bool Display the labels of the x and y axes.
showTitle bool Display the graph title.
showBorder bool Display a border around the graph.
borderWidth number Width of the border. (default: 2px)
showLegend bool Display a legend for the graph. (default: false)
xOffset number Offset x by a number of pixels. (default: 0px)
yOffset number Offset y by a number of pixels. (default: 0px)
Plots array an Array of Plot objects.
Graph.update() method Update the Graph.
Graph.set_offset(xoff,yoff) method Update the Graph's offset values.
Graph.drawBH() method Draws the background.
Graph.addPlot(Plot) method Adds a Plot to the Graph.
Graph.plotAll() method Draw all the Plots on the Graph.
