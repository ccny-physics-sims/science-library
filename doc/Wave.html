Class: Wave

Wave

new Wave(options)

Creates a Wave Object.
Parameters:
Name Type Description
options object Customize the default Wave.
Properties:
Name Type Description
bounds object Set the bounding box for the Wave. (default: {x: width, y: height})
theta number Angle of the Wave.
amplitude number Amplitude of the Wave.
wavelength number Wavelength of the wave. (default: 200px)
angularVelocity number Angular velocity of the wave.
displaySettings object Object containing display settings.
Properties
Name Type Description
line object Object containing display settings for a line drawn along the Wave.
Properties
Name Type Description
visible object Sets whether the line is visible.
dots object Object containing display settings for a series of dots drawn along the Wave.
Properties
Name Type Description
visible object Sets whether the dots are visible.
xSpacing number Determines the scaling of the wave along the x-axis.
yValues Array Array containing y-values of points along the wave.
Wave.updates() method Updates the Wave Object.
Wave.display() method Displays the Wave Object.

function setup() {
 frameRate(30);
