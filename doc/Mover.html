Class: Mover

Mover

new Mover(position, velocity, acceleration, mass, color)

Makes a little ball that can move. This object is to be replaced by KineticMass
Parameters:
Name Type Description
position p5.Vector The Mover's position.
velocity p5.Vector The Mover's velocity.
acceleration p5.Vector The Mover's acceleration.
mass number A scalar quantity indicating the mass.
color color The color of the Mover.
Properties:
Name Type Description
limit number The maximum speed of the mover.
tail bool A tail leaves little dots behind as it moves.
size number Default is set equivalent to mass.
outline color The color of the KineticMass' outline.
tail bool A tail leaves little dots behind as it moves.
tailFill color The color of the tail. (default: white)
tailStroke color The color of the tail's outline.
a number Angle. (default: 0)
aVelocity number Angular velocity. (default: 0)
aAcceleration number Angular acceleration. (default: 0)
Mover.update() method Update the Mover.
Mover.display() method Display the Mover.
Mover.giveItAnAcceleration() method Give the Mover an acceleration.
Mover.applyForce() method Apply a force to the Mover.
Mover.wrapEdges() method If the Mover goes off the side of the screen, it will reappear on the opposite side.
Mover.bounceEdges() method Bounces the Mover off the edges of the canvas.
Mover.towardMouse() method Make the Mover attracted to the user's mouse.
