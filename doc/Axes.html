Class: Axes

Axes

new Axes(options)

Creates a simple x-y axis.
Parameters:
Name Type Description
options object Customize the default properties. (Optional)
Properties:
Name Type Description
lineColor color The color of the lines. (default: black)
lineThickness number The thickness of the lines. (default: 1px)
rotateBy number Rotate axes by degrees or radians. (default: 0)
bounds object Set the bounds of the axes. (default: { minX: 0, maxX: 500, minY: 0, maxY: 500 })
Axes.display() method Displays the axes.

Examples
function setup() {
createCanvas(500, 500);
 myAxes = new Axes();
 myAxes.display();
}

createCanvas(500, 500);
var customSettings  = {lineColor : 'blue', lineThickness : 5}
myAxes = new Axes(customSettings);
myAxes.display();
}
