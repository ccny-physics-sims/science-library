Class: Plot

Plot

new Plot(pointArray, red, green, blue, weight, title)

Takes an array of Point objects and creates a plot of speed, velocity, and acceleration.
Parameters:
Name Type Description
pointArray array Array of Point objects to plot.
red color Color for representing speed.
green color Color for representing velocity.
blue color Color for representing acceleration.
weight number The weight.
title string Title of the graph.
Properties:
Name Type Description
data array The array of Points to be plotted. (default: pointArray)
pointSize number Size of the points. (default: 8px)
drawData bool Draw the data. (default: true)
plotTitle string Assign the title of the plot.
timeplot bool Decides if the time is plotted. (default: false)
Plot.plot(Graph) method Takes a Graph Object and returns a Plot.
Plot.getUser() method Get position of user's mouse.
Plot.tpRecord(variable, method Graph) Timeplotting function.
Plot.getPointDist(Point, method Point) Takes in two point objects and returns a new point object whose x and y values are the respective x and y distances.
Plot.fixChoord(scalex, method scaley, origin) Takes a p5.Vector as origin and scales it according to scalex and scaley.
