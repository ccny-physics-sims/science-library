Class: StaticBackground

StaticBackground

new StaticBackground(options)

Creates a static background.
Parameters:
Name Type Description
options object Customize the default properties for scenery. (Optional.)
Properties:
Name Type Description
type string Current options: 'basic' or 'grassy'
scenery.ground color Color of the foreground.
scenery.sky color Color of the sky.
scenery.vanishingPoint number Determines the location of the horizon as a portion of the total height. (.1 = 10% sky, .9 = 90% sky)
StaticBackground.display() method Draws the StaticBackground object.
Source:
  • background-static.js, line 1
Examples
// draws the default static background
function setup() {
 canvas = createCanvas(500, 500);
 bg = new StaticBackground();
 bg.display();
}
// draws a grassy background with some custom settings
function setup() {
  canvas = createCanvas(500, 500);
  bg = new StaticBackground({ type: "grassy", vanishingPoint: 0.2 });
  bg.display();
}
// change the sky color after creating the background
function setup() {
  canvas = createCanvas(500, 500);
  bg = new StaticBackground();
  bg.display();
}
function draw(){
  bg.scenery.sky = "orange";
  bg.display();
}
